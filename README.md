# Expense Tracker API

A backend expense tracker built using FastAPI.

## Features

- User Registration
- JWT Authentication
- Login System
- CRUD Operations
- SQLite Database
- SQLAlchemy ORM
- Protected Routes
- User-specific Expenses

## Technologies Used

- FastAPI
- SQLAlchemy
- SQLite
- JWT
- Pydantic

## Run Project

```bash
uvicorn app.main:app --reload
